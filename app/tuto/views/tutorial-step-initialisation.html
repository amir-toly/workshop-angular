L'objectif de cet atelier ets d'écrire une application permettant d'afficher et de filtrer des access logs d'un serveur web à l'aide du framework Angular

Vous trouverez ici la documentation officielle et là l'API de référence.

Dans un premier temps, pour initialiser l'application Angular, vous devez déclarer un module appelé workshop:

  • Allez dans le fichier app.js et déclarer un module angular.
  • Dans le fichier index.html, faites référence à ce module
