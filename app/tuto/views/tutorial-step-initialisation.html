L'objectif de cet atelier est de réaliser une application permettant d'afficher et de filtrer des access logs provenant d'un serveur Web à l'aide du framework AngularJS.

Le workshop est décomposé en plusieurs étapes, chacune d'elle validée par une ou plusieurs assertions exécutées à chaque rafraîchissement de page. Une fois l'étape passée avec succès, celle-ci est grisée.

Vous trouverez ici la documentation officielle et là l'API de référence.

Dans un premier temps, pour initialiser l'application AngularJS, il faut définir un module, on l'appelera workshop:

  • Dans le fichier workshop-angular/app/js/app.js, déclarer un module AngularJS
  • Dans le fichier workshop-angular/app/index.html, faire référence à ce module
