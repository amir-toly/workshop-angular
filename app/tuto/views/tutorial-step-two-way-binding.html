Le two-way data binding est une fonctionnalité du framework permettant de faire un lien bidirectionnel entre une variable du code JavaScript et un élément du de la vue. Chaque modification d'un côté ou de l'autre étant automatiquement répercuté. En d'autres termes, le famework effectue une synchronisation automatique entre le modèle et la vue.

L'objectif de cette étape est donc la mise en oeuvre de cette fonctionnalité. Pour se faire :

  • Dans le fichier workshop-angular/app/index.html, déclarer l'attribut ng-model avec la valeur query au niveau du champ de recherche
  • Afficher le contenu du champ dans la page
